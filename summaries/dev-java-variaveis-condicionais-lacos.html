Variáveis e Tipos de Dados

Tipagem Estática e Escopo

De acordo com Deitel, Java é fortemente tipada. Todas as variáveis devem ser declaradas com um tipo antes de serem utilizadas. Existem duas grandes categorias: Tipos Primitivos e Tipos de Referência.

Alerta de Edital (Pegadinha Clássica):

  • Variáveis de Instância/Classe: Recebem valores padrão (0, false, null).
  • Variáveis Locais: NÃO recebem valores padrão. Utilizar uma variável local sem inicialização prévia gera um Erro de Compilação (não é erro de execução!).

Estruturas Condicionais

Tomada de Decisão e Fluxo

Sintaxe: O perigo do Switch

int opcao = 2; switch (opcao) { case 1: System.out.println("Um"); case 2: System.out.println("Dois"); // Sem break! Ocorre o Fall-through case 3: System.out.println("Três"); } // Saída: "Dois", "Três"

As instruções de seleção (if, if...else, switch) determinam o fluxo com base em expressões booleanas.

Conceitos de Prova:

  • Operador Ternário (?:): Único operador em Java que recebe três operandos. Retorna um valor e frequentemente substitui um bloco if...else simples.
  • Fall-Through: No switch, se uma cláusula case coincidir e não houver um comando break, a execução "cairá" para os próximos cases independentemente de seus valores. A banca adora cobrar o resultado no console disso!
  • Strings no Switch: A partir do Java SE 7, é permitido usar objetos String nas expressões switch.
3

Estruturas de Repetição (Laços)

Iteração e Controle

WHILE

Pré-teste

A condição de continuação é testada antes da execução do corpo. Se a condição for falsa inicialmente, o corpo nunca será executado.

while (condicao) { // comandos }

FOR

Iteração Definida

Agrupa inicialização, condição e incremento no cabeçalho. Excelente para quando se sabe previamente o número de iterações.

for (int i=0; i<10; i++) { // comandos }

DO...WHILE

Pós-teste

Testa a condição após a execução do corpo. Garante que o bloco será executado pelo menos uma vez.

do { // comandos } while (condicao);

Controle de Fluxo Interno: Break vs Continue

Break: Quando executado em um `while`, `for`, `do...while` ou `switch`, causa a saída imediata dessa estrutura. A execução continua na instrução logo após a estrutura.

Continue: Pula as instruções restantes no corpo do laço para a iteração atual e prossegue para a próxima iteração do laço (no `for`, vai para a expressão de incremento).

Gabarito de Maratona
Característica Comportamento em Java
Promoção Inteira (Coerção) Tipos menores (`byte`, `short`, `char`) são promovidos a `int` antes de operações aritméticas.
Operadores Lógicos Curtos (`&&`, `||`) Avaliação de curto-circuito: A expressão à direita não é avaliada se o resultado já for determinado pela esquerda.
For Aprimorado (Enhanced `for`) Itera sobre arrays e coleções sem usar uma variável de contagem. Não permite modificar o array original durante a iteração.
Escopo de Variável no `for` Variáveis de controle declaradas dentro do cabeçalho do `for` não existem fora do bloco desse laço. Tentar acessá-las fora gera erro.
